❤️ Utility Helper — Test Page

Trang test UI cho extension. Click bong bóng ❤️ ở góc phải dưới → Hiện ⚙️ → Click ⚙️ mở Settings Panel

Đây là trang giả lập để test UI extension Utility Helper.
Các chức năng test: kéo thả heart bubble, toggle sub-bubbles, mở/đóng settings panel, chuyển đổi API.

Kéo ❤️ di chuyển → các bong bóng con sẽ theo sau mượt mà.
Click vào khoảng trống → thu bong bóng con lại.
